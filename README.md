## Keybindings Cheatsheet:

### Desktop Environment / Window Manager:
    - Meta (Win) + H : move focus left
    - Meta + J : move focus down
    - Meta + K : move focus up
    - Meta + L : move focus left
    - Shift + Alt - a : decrease window width
    - Shift + Alt - s : increase window height
    - Shift + Alt - w : decrease window height
    - Shift + Alt - d : increase window width
    - Alt + Space : activate application launcher 
    - Meta + Tab : walk through activities
    - Meta + Shift + Tab : reverse walk through activities
    - Alt + Q : close window
    - Ctrl + Alt + Esc : kill window
    - Alt + Up : show desktop grid
    - Ctrl + up : toggle overview
    - Alt + Left : switch one desktop left
    - Alt + Right : switch one desktop right
    - Meta + F1 : switch to desktop 1
    - Meta + F2 : switch to desktop 2
    - Meta + Alt + N : toggle night color

### Kitty:
    - Alt + Enter : new window 
    - Alt + D : close window
    - Alt + C : copy to clipboard
    - Alt + V : paste from clipboard
    - Ctrl + Shift + 1 : focus window 1
    - Ctrl + Shift + 2 : focus window 2

### Vimium:
    - ? : show the help dialog for a list of all available keys
